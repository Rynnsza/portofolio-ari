# Portofolio Ari

Website portofolio pribadi yang interaktif, modern, dan responsif. Project ini dibangun menggunakan HTML, CSS (SCSS), dan JavaScript dengan antarmuka yang didukung oleh framework Bootstrap, serta berbagai library untuk menghasilkan animasi dan fitur yang menarik.

## ðŸŒŸ Fitur Utama

* **Desain Responsif**: Tampilan yang optimal di berbagai ukuran layar (desktop, tablet, maupun smartphone).
* **Animasi Interaktif**: Menggunakan `WOW.js` dan `Animate.css` untuk animasi elemen saat di-scroll.
* **Teks Mengetik Dinamis**: Efek animasi teks mengetik otomatis menggunakan `Typed.js`.
* **Galeri Portofolio**: Dilengkapi dengan fitur filter kategori menggunakan `Isotope` dan tampilan popup gambar menggunakan `Lightbox`.
* **Slider/Carousel**: Menampilkan testimoni atau elemen bergeser menggunakan `Owl Carousel`.
* **Statistik Animasi**: Angka yang menghitung secara otomatis (counter) menggunakan `CounterUp` dan `Waypoints`.
